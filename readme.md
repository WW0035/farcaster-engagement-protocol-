# Farcaster Engagement Protocol

On-chain engagement and reward protocol for Farcaster, enabling cast-based quests, reputation scoring, and sybil-resistant airdrops.

## Features
- Social quests (like, follow, cast)
- Reputation scoring
- Merkle reward distribution
- Base network integration

## Stack
- Solidity
- Hardhat
- Base
- Farcaster API (Neynar)

## Setup

git clone https://github.com/yourname/farcaster-engagement-protocol.git
cd farcaster-engagement-protocol
npm install

## Deploy

npx hardhat run scripts/deploy.js --network base
